## 📌 About This Repository

Welcome to my **IoT & Robotics Project Vault**! 🚀

This repository is a comprehensive collection of **40+ hands-on projects** I have built (and will continue building) using:
- **Microcontrollers**: Arduino Uno, ESP8266 NodeMCU
- **Communication Modules**: HC-05 Bluetooth, Wi-Fi
- **Sensors**: Ultrasonic, PIR, DHT11, LDR, Soil Moisture, Rain Drop, Flame, MQ3 Gas, Pulse, Flex, Vibration, IR
- **Actuators**: L298N Motor Driver, DC Motors, Servo Motors, Relays
- **IoT Platforms**: Blynk IoT, MIT App Inventor
- **Simulation**: Tinkercad, Wokwi

Each project includes:
- ✅ Complete Arduino Code (`.ino` file)
- 🔌 Circuit Diagram / Wiring Instructions
- 📋 Component List
- 🎬 Working Demo Video (YouTube link)
- 📱 MIT App Inventor code (for Bluetooth/Wi-Fi projects)
